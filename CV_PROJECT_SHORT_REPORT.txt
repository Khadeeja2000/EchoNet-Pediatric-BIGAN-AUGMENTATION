PROJECT TITLE:
Temporal Super-Resolution for Echocardiogram Videos Using Optical Flow

================================================================================
1. WHAT WE DID
================================================================================

Implemented temporal super-resolution (frame interpolation) to increase video
frame rate from 30 fps to 60 fps using optical flow.

TECHNIQUE:
- Optical Flow (Farneback method)
- Frame Interpolation
- Forward/Backward Warping

================================================================================
2. HOW IT WORKS
================================================================================

STEP 1: Compute Optical Flow
- Analyzes motion between consecutive frames
- Calculates pixel-level motion vectors
- Uses Farneback dense optical flow algorithm

STEP 2: Interpolate Frames
- Creates intermediate frames between existing frames
- Uses forward warping (move pixels forward)
- Uses backward warping (move pixels backward)
- Blends both to create smooth intermediate frames

STEP 3: Generate Enhanced Video
- Inserts interpolated frames
- Increases frame rate: 30 → 60 fps (2x)
- Creates smoother motion
